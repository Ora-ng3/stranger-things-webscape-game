# Stranger Things Webscape Game !
I wanted to make an online game for a friend's birthday : a series of riddles based around Stranger Things. In the end I just made the riddles that I liked (so not linked to Stranger Things that much !), and it evolved into a more complex 3D game.
The game is composed of still views of a 3D scene, with transitions between them. As I wanted to keep it web-based and light, it made sense to prerender still images and transitions with Blender.
You can find all the assets, code and renders in this repo. Feel free to reuse things from the project, as long as you mention me (Amaury Chevoir).
Have fun playing it !
